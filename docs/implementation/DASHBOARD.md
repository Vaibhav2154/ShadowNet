# Dashboard (Next.js)

## Commands
```bash
cd web
pnpm install
pnpm dev        # http://localhost:3000
pnpm build
pnpm start
```

## Env
- `NEXT_PUBLIC_CONTROLPLANE_URL` must point to the control plane base URL

## Features
- Online peers list
- Handshake timestamps
- Bytes sent/received
- Endpoint details and connection state

## Config
- Tailwind CSS is used for styling; ESLint and TypeScript configured
